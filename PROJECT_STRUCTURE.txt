WHAT VERCEL RUNS:
=================

When you deploy on Vercel:

1. Reads package.json
2. Runs: npm install
3. Reads next.config.js
4. Runs: next build
5. Uploads build to Vercel servers
6. Ready to serve at: your-project.vercel.app

When user visits:
1. Vercel serves pages/index.js (React app)
2. User interacts with the UI
3. Browser calls pages/api/predict.js
4. Vercel serverless function runs
5. Returns prediction
6. UI displays result
